# SkillSwap - Frontend

This is the frontend of SkillSwap, built with React.js and Vite.

## Tech Stack

- React.js
- React Router
- Context API
- React-Bootstrap
- Axios
- Socket.io-client
- React-Toastify

## Setup

```bash
npm install
npm run dev
```

Runs on `http://localhost:5173`

## Environment Variables

Create a `.env` file in this folder:

```env
VITE_LOCALHOST = http://localhost:8000
VITE_SERVER_URL = http://localhost:8000
```
